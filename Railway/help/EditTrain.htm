Add train

To add a new train you must folow the steps bellow:

  1. If you wish to change it write the name of the train you would like to edit.
  2. If you wish to change it write the number of the wagons that the train you would like to edit has.
  3. If you wish to change it click on the button above the seat distribution chart that the train has. These represet seats in one wagon.
  4. Click on the save button and the window will appear with all of the parameters you have chosen.
  5. If the parameters displayed match the train you would like to edit click on the yes button. After conformation you will be taken to the page that displays all of the trains.
