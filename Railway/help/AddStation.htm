Add Station

To add a new station you must folow the steps bellow:

  1. Write a name of the new station in the field below station name.
  2. Click on the red dot, get the blue pin and drag it on the map where you want to add a new station.
  3. A window will appear showing you langitude and longitude of the place where you have placed the pin.
  4. If you think that longitude and latitude is correct, click the yes button.
  5. Click on the button that says add new station and the window will appear asking you are you sure.
  6. If you are sure click on the yes button. New station will be added and you will get back to the page where you can see all stations.

You cannot add two stations at once, you must add one station at the time. If you try to place two pins, the first one you placed wil dissapear.
