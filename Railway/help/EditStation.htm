Edit Station

To edit a station you must folow the steps bellow:

  1. If you wish to change station name write a new name of the station in the field below station name.
  2. If you wish to change the postion of the station click on the blue pin and drag it on the map where you want to place the station.
  3. A window will appear showing you langitude and longitude of the place where you have placed the pin.
  4. If you think that longitude and latitude is correct, click the yes button.
  5. Click on the button that says edit station and the window will appear asking you are you sure.
  6. If you are sure click on the yes button. The station will have new parameters and you will get back to the page where you can see all stations.
