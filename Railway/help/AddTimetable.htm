Add timetable

To add a new timetable folow the steps below:

  1. Chose a trainline from a dropdown list with trainline names.
  2. Chose a train from a dropdown list with train names.
  3. Chose a time when classic route starts from a dropdown list with times in "hh:mm" format.
  4. Chose a time when return route starts from a dropdown list with times in "hh:mm" format.
  5. Mark all of the days for which the new timetable applies.
  6. Click on the save button and a window will appear with all of the parameters you chose.
  7. If the parameters you see are the ones that you want the new timetable to have click on the yes button and after conformation you will be taken back to the page with all of the timetables.

Classic route and return route

Classic route is the route train takes from the first station to the last station, where as the return route is the when a train leaves the last station and returns back to the first one.

Classic route time is the time when train leaves the first station.

Return route time is the time when train leaves the last station to return back.
