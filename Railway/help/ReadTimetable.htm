List of timetables

On this page you are able to see all of the timetable available in the system. For each timetable name, train and days for which it applies are displays. You are also able to see times for which train will be in each station in the classic route and the return route. are displayed.

Adding a new timetable

To add a new timetable click on the add new timetable button.

Editing a timetable

To edit a timetable click on the button that says edit and it is located left of the name of the timetable you would like to edit.

Deleting a timetable

To delete a timetable click on the button that says delete and it is located left of the name of the timetable you would like to delete.

Classic route and return route

Classic route is the route train takes from the first station to the last station, where as the return route is the when a train leaves the last station and returns back to the first one.

Classic route time is the time when train leaves the first station.

Return route time is the time when train leaves the last station to return back.

Undo and redo

You can undo the last operation you have done by clicking on the undo button.

You redo the last operation you have undone by clicking on the redo button.
